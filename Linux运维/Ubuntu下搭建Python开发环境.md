# Ubuntu下搭建Python开发环境


## 安装Python3

    sudu apt-get remove python
    sudo apt-get install python3

## 安装pip

    sudo apt install python3-pip
    
    常用命令
    pip3 --version
    pip install scrapy
    pip uninstall scrapy
    pip search "search_query"
    pip list

## 安装开发工具

	sudo apt install build-essential python3-dev  python3-setuptools
